Для начала поговорим о разнице между HTML-кодом и DOM. Например, обычный элемент — это, очевидно, HTML-код.
Этот элемент можно использовать в html-файлах, у него есть набор атрибутов, который определяет внешний вид и поведение
создаваемой с его помощью таблицы. Собственно говоря, сам по себе тег
не имеет никакого отношения к JavaScript. Связь между HTML-элементами, присутствующими в документе, и JavaScript-кодом,
обеспечивает DOM (Document Object Model, объектная модель документа). DOM даёт возможность взаимодействовать
с HTML-элементами из JavaScript-кода так, как будто они являются объектами.
